TableManager

interface TableManager<K, V, T : Entity<*>>
Content copied to clipboard

A table manager is a manager for ID tables, thats uses key-value pair to stores their data. K refers to the primary key used to get a value. V refers to the value of a key. T refers to the entity model of this table.
